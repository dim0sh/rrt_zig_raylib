# Content
zig implementation of RRT based pathfinding
# Usage
run command:
```bash
zig build run
```
this command builds and executs the project
# Functionality
## vizualization of algorithms:
- RRT
- TRRT
# Requirements
zig language is required to run.

version used 0.13.0
